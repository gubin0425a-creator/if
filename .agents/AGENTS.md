# Project Rules

- **버전 미지정 시 동시 업데이트 및 검증**:
  사용자가 명시적으로 특정 버전을 언급하지 않은 모든 기능 변경 및 UI 업데이트 요청에 대해서는 **PC 버전(`ui/index.html`)과 모바일 버전(`ui/mobile.html`) 둘 다 자동으로 빌드/업데이트**를 진행하고 시뮬레이션 및 검증해야 합니다.

- **독립 구동을 위한 클라우드 배포 규칙**:
  모바일 앱이 로컬 PC 전원 차단과 관계없이 24시간 독립적으로 동작할 수 있도록, 향후 백엔드 코드는 클라우드 환경(예: Render, Railway, AWS 등)으로의 배포 및 가동을 지원하며 모바일 앱이 클라우드 주소로 통신할 수 있는 복구/설정 화면을 제공합니다.
